# Bundled font attribution

sigil embeds a small set of FIGlet fonts (`.flf`). They are trimmed to the 102
required characters (95 printable ASCII + the 7 German characters) to keep the
binary small and to avoid code-tagged glyphs that some parsers reject.

| File | Font | Author(s) |
|------|------|-----------|
| (built-in) | Standard | Glenn Chappell & Ian Chai |
| ANSI_Shadow.flf | ANSI Shadow | contributed FIGlet font |
| Slant.flf | Slant | Glenn Chappell |
| Big.flf | Big | Glenn Chappell |
| Small.flf | Small | Glenn Chappell |
| Doom.flf | Doom | Frans P. de Vries |
| Bloody.flf | Bloody | contributed FIGlet font |
| 3D-ASCII.flf | 3D-ASCII | contributed FIGlet font |
| ANSI_Regular.flf | ANSI Regular | contributed FIGlet font |
| Ghost.flf | Ghost | contributed FIGlet font |

FIGlet and the classic FIGlet fonts are distributed under permissive terms that
allow free use and redistribution. See <http://www.figlet.org/> for the FIGlet
project and font license details.
